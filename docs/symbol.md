


## Variable names and meaning
To consolidate variable names and terminology for developers.

|  ``variable_name``   |  ``Meaning``  |
|:--------------------:|:-------------:|
|      ``nr``       |     density in real space    |
|      ``ng``       |  density in reciprocal space |
|      ``cg``       |  Hermitian coefficient for plane wave basis |
|     ``w``   |  trainable parameters in wave function objects |
|      ``r_grid``    |  a grid in real space |    
|      ``g_grid``    |  a grid in reciprocal space |
|     ``mask``       | the mask for g_grid that enforces the cut-off energy |
|     ``k_grid``     |  a grid in the first Brillouin zone  |
|     ``k_points``  |   k points sampled for band structure |
|     ``A``         | crystal constant  |
|     ``B``         | inverse crystal constants |
|    ``occ``        |  occupation mask  |
